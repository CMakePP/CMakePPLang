cmakepp_lang.types/path

cmakepp_lang.types/path.cpp_is_path(_ip_result _ip_str2check)

Determines if a string is lexically convertible to an absolute filepath.

This function wraps a call to CMake’s if(IS_ABSOLUTE ...) in order to determine if the provided string is lexically convertible to an absolute filepath.

Parameters:

  • _ip_result (identifier) – An identifier to hold the result.

  • _ip_str2check – The string which may be an absolute filepath.

Returns:

TRUE if _ip_str2check is an absolute filepath and FALSE otherwise. The result is returned via _cip_result.

Return type:

bool

Error Checking

cpp_is_path will assert that the caller has provided exactly two arguments. If this is not the case an error will be raised.

Example Usage:

The following code snippet checks whether ${CMAKE_BINARY_DIR} is a relative directory (it always is by default):

include(cmakepp_lang/types/filepath)
cpp_is_path(result "${CMAKE_BINARY_DIR}")
message("Is a filepath: ${result}")  # Prints TRUE
