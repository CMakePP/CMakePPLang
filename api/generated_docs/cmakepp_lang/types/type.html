cmakepp_lang.types/type

cmakepp_lang.types/type.cpp_is_type(_it_result _it_type)

Determines if a string is lexically convertible to a type literal.

CMakePP defines a series of type literals for types recognized by CMake and for types recognized by CMake and CMakePP. Additionally, users can create classes to add even more type literals. This function will determine if an identifier is also a type literal.

Parameters:

  • _it_result (desc) – Name for the variable which will hold the result.

  • _it_type (str) – The object for which we want to know if it is a type literal.

Returns:

_it_result will be set to TRUE if _it_type is a type and FALSE otherwise.

Return type:

bool
