cmakepp_lang.class/ctor

cmakepp_lang.class/ctor.cpp_class_ctor(_cc_this _cc_type)

Handles the construction of a class instance.

This function handles the construction of a new class and calls to the constructors for base classes.

Parameters:

  • _cc_this (desc) – The handle to the object that will be constructed.

  • _cc_type (class) – The class we are calling the constructor for.

  • *args – The arguments the constructor was called with (not including the instance name and class).

Returns:

_cc_this will be set to the mangled name of the newly constructed instance.

Return type:

desc
